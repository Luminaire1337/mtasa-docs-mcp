# Security Policy

## Supported Versions

We provide security fixes for the latest stable release.

| Version | Supported |
| ------- | --------- |
| 1.x     | Yes       |
| < 1.0.0 | No        |

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Report security issues via private Discord message to `@matchlesslumi` with:

- A clear description of the issue and potential impact
- Reproduction steps or proof-of-concept details
- Affected versions and environments

What you can expect:

- Initial response within 72 hours
- Ongoing status updates during triage and remediation
- Credit in release notes (optional) after fix disclosure

If your report is accepted, we will coordinate responsible disclosure timing
before public release of technical details.
